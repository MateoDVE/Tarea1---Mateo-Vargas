Nro Elemento Proposito Ejemplo
1 h1 Define un encabezado principal en la página. Es el nivel más alto de encabezado y suele usarse para títulos importantes. <h1>Encabezado Principal</h1>
2 h2 Encabezado de segundo nivel, usado generalmente para subtítulos o secciones dentro del contenido. <h2>Encabezado Secundario</h2>
3 p Define un párrafo de texto. Es el elemento básico para escribir contenido en HTML. <p>Este es un párrafo.</p>
4 span Elemento en línea que se usa para aplicar estilos o manipular partes específicas de un texto. <span style="color:red;">Texto en rojo</span>
5 div Un contenedor genérico en bloque utilizado para agrupar contenido y aplicar estilos o scripts. <div>Este es un contenedor</div>
6 header Representa la cabecera de una sección o de todo el documento. Suele contener logotipos, menús de navegación y títulos. <header>Contenido del encabezado</header>
7 footer Define el pie de página de un documento o sección, normalmente con información de contacto, derechos de autor o enlaces adicionales. <footer>Derechos reservados 2024</footer>
8 nav Define una sección de navegación que contiene enlaces a otras partes del sitio web. <nav><a href="index.html">Inicio</a></nav>
9 section Define una sección dentro de un documento que agrupa contenido temáticamente relacionado. <section>Contenido de la sección</section>
10 article Representa contenido independiente y autocontenido, como un artículo de blog o una noticia. <article>Noticia importante</article>
11 ul Define una lista desordenada, comúnmente con viñetas. <ul><li>Elemento 1</li><li>Elemento 2</li></ul>
12 ol Define una lista ordenada, comúnmente numerada. <ol><li>Paso 1</li><li>Paso 2</li></ol>
13 li Representa un elemento dentro de una lista. <li>Elemento de lista</li>
14 table Define una tabla para organizar datos en filas y columnas. <table><tr><td>Dato</td></tr></table>
15 tr Representa una fila dentro de una tabla. <tr><td>Celda</td></tr>
16 td Define una celda de datos dentro de una tabla. <td>Contenido</td>
17 th Define una celda de encabezado dentro de una tabla. <th>Encabezado</th>
18 form Define un formulario para la entrada de datos del usuario. <form><input type="text"></form>
19 em El elemento em se utiliza para marcar un texto que se desea enfatizar de manera leve, usualmente se muestra en cursiva, aunque su propósito principal es indicar que el contenido tiene una relevancia adicional. Es comúnmente usado en situaciones donde se quiere destacar un término, acción o frase importante dentro de un bloque de texto. El uso de em permite a los motores de búsqueda y a los lectores de pantalla comprender que ese texto es importante, lo que lo hace útil tanto para la accesibilidad como para la semántica del documento. <em>Texto en cursiva</em>
20 mark El elemento mark se usa para resaltar partes del texto, generalmente con un fondo de color amarillo o similar, para indicar que esa parte del contenido es relevante o importante dentro de un contexto específico. Es útil en aplicaciones como los motores de búsqueda, donde las palabras clave o frases importantes pueden ser destacadas para facilitar la lectura y comprensión. Esta etiqueta también ayuda a mejorar la experiencia del usuario en términos de visibilidad de información clave dentro de un documento. <mark>Texto resaltado</mark>
21 q El elemento q se utiliza para envolver una cita corta o breve dentro de una línea de texto. Esta etiqueta se usa cuando se citan palabras o frases de otras fuentes o personas y el contenido de la cita generalmente está en línea con el texto circundante. Los navegadores suelen agregar automáticamente comillas alrededor del contenido de la etiqueta q, facilitando la lectura y comprensión de que se trata de una cita. <q>Cita corta</q>
22 blockquote El elemento blockquote se usa para citar un bloque de texto largo o de varias líneas de una fuente externa, normalmente se usa para marcar citas textuales extensas. Este elemento se suele mostrar con sangrías en los lados o un estilo específico que lo diferencia del texto regular, ayudando a los usuarios a identificar fácilmente las citas largas. Su uso mejora la estructura semántica del documento y facilita la comprensión del contenido citado. <blockquote>Cita larga</blockquote>
23 cite El elemento cite se utiliza para hacer referencia a la fuente de una cita o a una obra en particular. Su uso indica que el texto dentro de la etiqueta es el título de una obra, una referencia literaria o una fuente de información. Es útil en contextos académicos, literarios o cuando se quiere dar crédito a la autoría de una obra específica, mejorando la semántica y accesibilidad del contenido. <cite>Cita</cite>
24 dfn El elemento dfn se utiliza para marcar una definición o término que se introduce por primera vez en un documento. Es útil para indicar que el término está siendo definido, lo que mejora la semántica del contenido. Normalmente, los navegadores muestran este texto en cursiva para hacerlo destacar, aunque esto puede variar según el estilo. <dfn>Definicion</dfn>
25 abbr El elemento abbr se utiliza para representar una abreviatura o sigla en un documento. Este elemento proporciona una forma semántica de mostrar abreviaturas y permite incluir una descripción completa del término a través del atributo title. Esto es útil para los usuarios que no están familiarizados con la abreviatura, ya que pueden ver el significado completo cuando pasan el cursor sobre el texto. <abbr>Abreviatura</abbr>
26 small El elemento small se utiliza para reducir el tamaño del texto en comparación con el texto circundante. Se suele usar para mostrar detalles o información adicional que no es central para el contenido principal, como notas al pie, advertencias o términos legales. Aunque su tamaño se reduce visualmente, su uso es importante para mantener la semántica adecuada dentro del documento. <small>Texto pequeño</small>
27 ins El elemento ins se utiliza para marcar un texto que ha sido insertado o agregado en el documento. Generalmente se usa en el contexto de cambios o ediciones dentro de un documento, donde se quiere resaltar el nuevo contenido. Los navegadores suelen mostrar este texto con un subrayado para indicar que es nuevo. <ins>Texto insertado</ins>
28 del El elemento del se utiliza para marcar un texto que ha sido eliminado del documento. Este elemento se usa a menudo en el contexto de ediciones de contenido, mostrando que una parte del texto ya no es relevante o ha sido suprimida. Los navegadores suelen mostrar este texto tachado para indicar su eliminación. <del>Texto eliminado</del>
29 s El elemento s se utiliza para marcar texto que ha sido tachado o que ya no es válido. A diferencia del elemento del, que implica una eliminación activa, s se usa para representar cambios o correcciones que no necesariamente indican una eliminación definitiva, pero que visualmente muestran el texto como tachado. <s>Texto tachado</s>
30 code El elemento code se utiliza para representar fragmentos de código en línea. Se usa comúnmente en documentos técnicos o tutoriales para denotar una línea de código o comandos que deben ser ejecutados. Este texto se suele mostrar en un tipo de letra monoespaciada para diferenciarlo del texto normal. <code>Codigo</code>
31 samp El elemento samp se usa para mostrar ejemplos de salida de programas o sistemas informáticos. Este texto también suele mostrarse en una fuente monoespaciada y se utiliza comúnmente en documentación técnica o tutoriales para representar lo que un usuario podría ver al ejecutar un programa o comando. <samp>Ejemplo de codigo</samp>
32 kbd El elemento kbd se utiliza para representar la entrada de teclado, es decir, cuando se quiere mostrar qué tecla o combinación de teclas debe presionar un usuario. Este texto generalmente se muestra en una fuente monoespaciada para hacerlo más legible y diferenciado de otro texto. <kbd>Tecla de teclado</kbd>
33 pre El elemento pre se utiliza para mostrar texto preformateado, es decir, texto donde se conserva el formato de espaciado y saltos de línea tal como se ha escrito en el código fuente. Esto es útil cuando se quiere presentar contenido como código o texto que debe mantener un formato específico. <pre>Texto preformateado</pre>
34 address El elemento address se utiliza para proporcionar información de contacto o dirección dentro de un documento. Es comúnmente utilizado en el pie de página o en la sección de contacto de una página web, y puede incluir dirección postal, correo electrónico o número de teléfono. <address>Direccion</address>
35 character entity reference La referencia de entidad de caracteres &lt; se utiliza para representar el carácter "<" en un documento HTML. Las referencias de entidades de caracteres son útiles para representar caracteres especiales que de otro modo podrían ser interpretados como código HTML, como los signos de menor y mayor que. &lt;
36 ul El elemento ul define una lista desordenada, comúnmente con viñetas. Se usa cuando se quiere agrupar elementos que no tienen un orden específico, como una lista de tareas o características. <ul>Lista desordenada</ul>
37 li El elemento li se usa para representar un elemento dentro de una lista, ya sea ordenada (ol) o desordenada (ul). Cada elemento de lista se marca con esta etiqueta. <li>Elemento de lista</li>
38 table El elemento table se utiliza para definir una tabla. Las tablas son útiles para organizar datos en filas y columnas, permitiendo presentar la información de manera estructurada y clara. <table>Tabla</table>
39 src El atributo src se utiliza en elementos como img o script para especificar la ubicación de la fuente de un archivo, como una imagen o un script. Este atributo es esencial para cargar contenido externo en una página web. <img src="imagen.jpg">
